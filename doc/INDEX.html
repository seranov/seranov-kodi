File Format Notes

#### Markdown Files - Naming: UPPERCASE.md or UPPERCASE.LANG.md (e.g., README.ru.md) - Encoding: UTF-8 - Line endings: LF (Unix-style) - Automatically converted to HTML by GitHub Actions

#### HTML Files (Generated) - Auto-generated from markdown files - Located: docs/doc/ directory on GitHub Pages - Updated: Every deployment - Not stored in git repository
