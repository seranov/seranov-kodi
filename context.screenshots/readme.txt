Bidon Popup Screenshots - Kodi Add-on
======================================

Version: 1.0.5
Author: seranov
Email: seranov@yandex.ru
Website: https://github.com/seranov/kodi-play-random

Description
-----------
This add-on provides a context menu item that allows you to view screenshots and images
in a slideshow format. Optionally, it can also display file_id.diz file text content.

Features
--------
- Context menu integration
- Slideshow of local images
- Support for file_id.diz text files
- Simple and lightweight

Installation
------------
1. Download the add-on zip file
2. In Kodi, go to Add-ons -> Install from zip file
3. Select the downloaded zip file
4. The add-on will be installed and ready to use

Usage
-----
1. Navigate to a video or media item in Kodi
2. Open the context menu (usually 'C' key or right-click)
3. Select "Popup Screenshots" from the menu
4. Enjoy the slideshow!

Requirements
------------
- Kodi 19.x (Matrix) or higher
- Python 3.x
- script.module.kodi-six
- script.module.routing
