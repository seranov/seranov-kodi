# Installation Guide / Руководство по установке

## English

### Quick Start

The easiest way to install all add-ons from this repository is to install the repository add-on first.

### Option 1: Install Repository Add-on (Recommended)

1. **Download the repository zip file:**
   - Go to [Releases](https://github.com/seranov/kodi-play-random/releases/latest)
   - Download `repository.seranov-1.0.0.zip`

2. **Enable "Unknown sources" in Kodi:**
   - Open Kodi **Settings** (gear icon)
   - Go to **System** → **Add-ons**
   - Enable **Unknown sources**
   - Click **Yes** to confirm the warning

3. **Install the repository:**
   - Go to **Add-ons** menu (puzzle piece icon)
   - Click the **box icon** at the top left
   - Select **Install from zip file**
   - Navigate to where you downloaded `repository.seranov-1.0.0.zip`
   - Select the file and wait for the "Add-on installed" notification

4. **Install add-ons from the repository:**
   - Go to **Add-ons** menu
   - Click the **box icon** at the top left
   - Select **Install from repository**
   - Select **Seranov's Kodi Repository**
   - Choose the category (Video add-ons, Context menus, Services)
   - Select the add-on you want to install
   - Click **Install**

### Option 2: Add Repository Source (For Auto-Updates)

1. **Add file source:**
   - Go to **Settings** → **File manager**
   - Click **Add source**
   - Click **<None>**
   - Enter the URL: `https://raw.githubusercontent.com/seranov/kodi-play-random/main/repo/`
   - Enter a name: `Seranov Repository`
   - Click **OK**

2. **Install repository from source:**
   - Go to **Add-ons** → **Install from zip file**
   - Select **Seranov Repository** source
   - Select `repository.seranov-1.0.0.zip`
   - Wait for installation confirmation

3. **Install add-ons:**
   - Go to **Add-ons** → **Install from repository**
   - Select **Seranov's Kodi Repository**
   - Browse and install desired add-ons

### Option 3: Install Individual Add-on

If you only want one specific add-on without the repository:

1. Download the specific add-on zip from [Releases](https://github.com/seranov/kodi-play-random/releases/latest)
2. Follow steps 2-3 from Option 1 to install the zip file

**Note:** Without installing the repository, you won't receive automatic updates.

## Русский

### Быстрый старт

Самый простой способ установить все дополнения из этого репозитория - сначала установить дополнение-репозиторий.

### Вариант 1: Установка дополнения-репозитория (Рекомендуется)

1. **Скачайте zip-файл репозитория:**
   - Перейдите в [Релизы](https://github.com/seranov/kodi-play-random/releases/latest)
   - Скачайте `repository.seranov-1.0.0.zip`

2. **Включите "Неизвестные источники" в Kodi:**
   - Откройте **Настройки** Kodi (значок шестеренки)
   - Перейдите в **Система** → **Дополнения**
   - Включите **Неизвестные источники**
   - Нажмите **Да** для подтверждения предупреждения

3. **Установите репозиторий:**
   - Перейдите в меню **Дополнения** (значок пазла)
   - Нажмите на **иконку коробки** в верхнем левом углу
   - Выберите **Установить из файла zip**
   - Перейдите к месту, куда вы скачали `repository.seranov-1.0.0.zip`
   - Выберите файл и дождитесь уведомления "Дополнение установлено"

4. **Установите дополнения из репозитория:**
   - Перейдите в меню **Дополнения**
   - Нажмите на **иконку коробки** в верхнем левом углу
   - Выберите **Установить из репозитория**
   - Выберите **Seranov's Kodi Repository**
   - Выберите категорию (Видеодополнения, Контекстные меню, Службы)
   - Выберите дополнение, которое хотите установить
   - Нажмите **Установить**

### Вариант 2: Добавление источника репозитория (Для автообновлений)

1. **Добавьте источник файлов:**
   - Перейдите в **Настройки** → **Диспетчер файлов**
   - Нажмите **Добавить источник**
   - Нажмите **<Нет>**
   - Введите URL: `https://raw.githubusercontent.com/seranov/kodi-play-random/main/repo/`
   - Введите имя: `Seranov Repository`
   - Нажмите **OK**

2. **Установите репозиторий из источника:**
   - Перейдите в **Дополнения** → **Установить из файла zip**
   - Выберите источник **Seranov Repository**
   - Выберите `repository.seranov-1.0.0.zip`
   - Дождитесь подтверждения установки

3. **Установите дополнения:**
   - Перейдите в **Дополнения** → **Установить из репозитория**
   - Выберите **Seranov's Kodi Repository**
   - Просмотрите и установите нужные дополнения

### Вариант 3: Установка отдельного дополнения

Если вам нужно только одно конкретное дополнение без репозитория:

1. Скачайте конкретное дополнение zip из [Релизов](https://github.com/seranov/kodi-play-random/releases/latest)
2. Следуйте шагам 2-3 из Варианта 1 для установки zip-файла

**Примечание:** Без установки репозитория вы не будете получать автоматические обновления.

## Troubleshooting / Устранение неполадок

### English

**Problem:** "Install from zip file" option is greyed out

**Solution:** You need to enable "Unknown sources" in Settings → System → Add-ons

**Problem:** Add-on installation fails

**Solution:** 
- Make sure you're using Kodi 19 (Matrix) or later
- Check that all dependencies are available
- Look at the Kodi log file for detailed error messages

**Problem:** Repository doesn't show any add-ons

**Solution:**
- Make sure you have an internet connection
- Try updating the repository: Right-click on "Seranov's Kodi Repository" and select "Check for updates"

### Русский

**Проблема:** Опция "Установить из файла zip" неактивна

**Решение:** Необходимо включить "Неизвестные источники" в Настройки → Система → Дополнения

**Проблема:** Установка дополнения не удается

**Решение:**
- Убедитесь, что вы используете Kodi 19 (Matrix) или новее
- Проверьте, что все зависимости доступны
- Посмотрите лог-файл Kodi для подробных сообщений об ошибках

**Проблема:** Репозиторий не показывает никаких дополнений

**Решение:**
- Убедитесь, что у вас есть подключение к интернету
- Попробуйте обновить репозиторий: Нажмите правой кнопкой на "Seranov's Kodi Repository" и выберите "Проверить обновления"
